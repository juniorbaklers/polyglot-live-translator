# Guide utilisateur

1. Démarrer l'application Windows.
2. Choisir le microphone, le son de l'ordinateur ou l'onglet du navigateur.
3. Choisir les langues source et cible.
4. Pour le navigateur, saisir dans l'extension le code affiché par l'application.
5. Cliquer sur « Capturer le son de cet onglet ».
6. Arrêter immédiatement depuis l'extension ou l'application.
7. Retrouver la session dans « Transcriptions » et choisir un export.

Pour une démonstration sans service en ligne, activer le mode démo.
