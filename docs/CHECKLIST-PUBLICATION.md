- [x] Code source Windows, extension et contrats partagés
- [x] README, licence, sécurité et contribution
- [x] Mode démonstration
- [x] Contrôles TypeScript
- [x] Extension compilée
- [x] Workflow de compilation Windows
- [ ] Premier workflow Windows réussi sur GitHub
- [ ] Test du microphone et de WASAPI sur Windows 10/11
- [ ] Test de l'extension sur Chrome et Edge
- [ ] Ajout des captures d'écran réelles
- [ ] Signature numérique des installateurs de production

Les cases restantes nécessitent l'exécution sur une machine Windows ou sur
GitHub Actions après le téléversement du projet final.
